Feedback

Tijdens het maken van verschillende opdrachten en projecten heb ik veel feedback gevraagd. Deze feedback heb ik verwerkt in mijn producten. Hierbij heb ik gecommuniceerd met docenten, onze opdrachtgever en diverse andere mensen waarvan ik feedback kon krijgen.

Communication

Feedback vanuit docenten

Een groot deel van de feedback die ik heb gekregen komt vanuit docenten. Zo heb ik bijvoorbeeld voor mijn portfolio veel feedback gekregen, zowel uit de portfolio reviews als zelf gevraagde feedback. Een ander voorbeeld is dat ik veel feedback op mijn sprint X project heb gevraagd, zowel door mijn begeleider als andere docenten. Deze feedback heb ik voor een groot deel ook in feedpulse gezet.

Feedback opdrachtgever

Als groep Eternia 1 hebben we onze opdrachtgever Ruby Mus veel feedback gevraagd op de producten die we geleverd hebben. Zo hebben we onze eerste versies van verschillende met Ruby Mus gedeeld, om feedback te krijgen zodat we deze snel kunnen toepassen. Ook hebben we, bijvoorbeeld voor het logo, veel designs tegelijk gemaakt, en Ruby Mus gevraagd welke haar het meeste aanspreekt. Zo konden we een goed beeld krijgen waar Ruby Mus naar zocht.

Diverse feedback

Naast feedback vanuit docenten en onze opdrachtgever, heb ik ook feedback gekregen van andere bronnen. Zo heb ik vaak aan mijn medestudenten gevraagd wat ze van verschillende producten of keuzes vinden. We hebben veel binnen onze groep Eternia 1 feedback gevraagd, zowel voor producten voor onze opdrachtgever als persoonlijke producten. Verder heb ik ook af en toe simpele feedback heb gekregen vanuit vrienden of familie, vooral wat ze vinden van het uiterlijk van de producten.

Wat heb ik geleerd

Ik heb geleerd om genoeg feedback te vragen om zo mijn producten beter te maken.

Wat kon beter

Ik had nog meer feedback kunnen vragen aan de docenten om nog beter mijn producten te verbeteren.
