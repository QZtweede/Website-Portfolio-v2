Mediaverkenning

In sprint 0 hebben we als groep een mediaverkenning gemaakt voor onze opdrachtgever Ruby Mus. Een groot deel hiervan was onderzoek doen, maar daarentegen was het ook belangrijk dat het document er aansprekelijk uit zag. Ik heb designs gemaakt voor meerdere paginas in dit document.

Klik hier om het document te zien

Design

Kleurenpalet

Op basis van de voorkeuren van onze opdrachtgever Ruby Mus hebben we een kleurenpalet gemaakt. Deze gaan we als groep gebruiken voor zowel de mediaverkenning als andere producten die we voor Ruby Mus gaan voorleggen. Ruby Mus heeft voor ons duidelijk gemaakt dat de focus van dit pallet op rood moet liggen. Wel werd de nadruk gelegd op dat het niet moet lijken op een horror thema. Om te zorgen dat dit niet gebeurt hebben we ook tinten van blauw en paars toegevoegd. Verder hebben we voor de details goud gebruikt, ook omdat Ruby Mus dit ook aantrekkelijk vond.

Inspiratie

De eerste pagina die ik voor de mediaverkenning heb gedesigned is de inspiratie pagina. Hierin staan de artiesten waarvan we inspiratie hebben opgedaan voor verschillende delen van onze producten. Voor het design van deze pagina heb ik gekeken naar verschillende tijdschrift pagina's. Hier zie je vaak dat er een design word verdeeld over twee paginas. Dit heb ik geprobeerd na te maken door een groot design te maken in plaats van twee aparte. Verder heb ik het kleurenpalet gebruikt. Rood, blauw en zwart zitten in de achtergrond, wit en goud word gebruikt voor de tekst.

Media landschap

Verder heb ik de pagina voor het media landschap gemaakt. Deze heb ik wel door twee gedeeld. Op deze pagina's staan suggesties voor social media platformen die Ruby Mus kan gebruiken. Deze paginas zijn meer gefocust op informatie, dus heb ik het design hierop aangepast. Zo heb ik veel ruimte voor tekst en afbeeldingen gemaakt. Daarentegen heb ik wel gelet op het kleurenpalet, dus zijn de kleuren wel verwerkt in deze pagina. Om de tekst goed leesbaar te maken heb ik blokken gebruikt waarin de tekst staat.

Wat heb ik geleerd

Ik heb geleerd hoe ik mijn design meer als tijdschrift pagina's kan weergeven. Verder heb ik meer ervaring in de tool Photoshop opgedaan.

Wat kon beter

Ik had me meer kunnen focussen op het design van de media landschap pagina. Zo is de pagina nu erg gefocust op de informatie.
