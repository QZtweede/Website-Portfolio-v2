Portfolio IM

Een groot deel van dit semester is het portfolio. Hiervoor heb ik verschillende prototypes gemaakt, en feedback op mijn proces gekregen bij de portfolio reviews. In dit project laat ik mijn proces van het interactieve media gedeelte van mijn portfolio zien.

Portfolio review 2

Bij portfolio review 2 was mijn portfolio al meer een geheel geworden. Daarentegen kreeg ik als feedback dat ik beter bronnen en feedback kon verwerken in mijn tekst. De site was op andere beeldschermen niet helemaal hetzelfde, dus moest ik ook naar responsiveness kijken. Verder was de unieke stijl goed uitgewerkt, maar kreeg ik ook te horen dat er nog veel content toegevoegd moest worden. Deze feedback heb ik meegenomen, en ik lag nog steeds redelijk goed op schema.

Portfolio review 3 / 4

In portfolio review 3 kreeg ik als feedback dat het handig was om een "wat kon beter" deel toe te voegen aan mijn projecten. Verder is de layout en stijl origineel en goed naar voren gebracht. Daarentegen was de navigatie onhandig omdat er niet snel van leeruitkomst veranderd kan worden, en liep ik achter met het vullen van het portfolio met content. Hierdoor liep ik bij portfolio review 3 achter. Dit motiveerde mij om hard te werken om de feedback te verwerken, en een 4e review aan te vragen. In de vierde review was mijn portfolio al een stuk beter, en moesten alleen de laatste details worden toegevoegd.

Wat heb ik geleerd

Ik heb geleerd om feedback te verwerken in mijn interactieve media product. Daarnaast heb ik geleerd om mijn eigen stijl op een goede manier in de website te verwerken, en hiermee te experimenteren.

Wat kon beter

Ik had mijn feedback sneller kunnen verwerken om zo beter mijn werk te kunnen verdelen. Ook zou ik dan meer feedback kunnen vragen op mijn portfolio.
