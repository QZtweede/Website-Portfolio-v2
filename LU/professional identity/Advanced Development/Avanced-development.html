Advanced development

In dit project heb ik onderzoek gedaan naar verschillende javascript libraries. Hiervan heb ik drie uitgewerkt in een demo. Uiteindelijk heb ik een hiervan ook in mijn portfolio verwerkt. Dit project was een van mijn eerste ervaringen met javascript libraries.

Development / PI

Anime.js

Anime.js is een library gemaakt voor animaties. De mogelijkheden met anime.js zijn erg breed, omdat de library veel verschillende functies en eigenschappen heeft. Zie hier. Ik was geïnterreseerd in deze library omdat deze makkelijk te begrijpen was, en zo dus gebruikt kan worden voor grote of kleine animaties.

Demo

Granim.js

Granim.js is een library om makkelijk kleur overlopen te maken. De library is vergeleken met anime.js een stuk simpeler, maar ook meer gefocussed op een functie. De library is simpel te begrijpen, en kan toegepast worden in verschillende omstandigheden. Zelf heb ik geëxperimenteerd om deze library te implementeren in mijn portfolio, dit vond ik daarentegen niet erg bij de stijl passen.
Github
Demo

AOS.js

AOS is een erg simpele library die gefocussed is op simpele animaties gebaseerd op scrollen. De library is heel makkelijk te gebruiken, vooral omdat alles via HTML word toegepast. Daarentegen lijken de animaties erg professioneel, en geven dezen een extra touch aan je website. Zelf heb ik geëxperimenteerd om de library te gebruiken in mijn portfolio. Het resultaat vond ik aansprekend, dus heb ik het toegepast op meerdere pagina's.
