Website Ruby Mus

In dit semester heb ik veel tijd besteed aan onze werkgever Ruby Mus. Een groot deel hiervan is de website die we voor haar gemaakt hebben. De website is gemaakt in HTML, CSS en javascript.

Klik hier om de website te zien Github

Development

Template

Als eerst heb ik een simpele template gemaakt voor de website. Dit heb ik gedaan zodat mijn groepsleden apart aan hun eigen delen kunnen werken, zonder dat de stijl te veel verschilt. Ik heb vast de containers gemaakt waar de content in komt. Verder hebben we taken verdeeld. Ik werk vooral aan de feed pagina, waar de recente posts op social media van Ruby Mus worden weergegeven. Ook werk ik aan de muziek pagina, waar we in dit geval haar nieuwste lied laten horen.

Feed

In de feed pagina staan de recente posts van Ruby Mus. Ze maakt vooral gebruik van Facebook en Instagram, dus heb ik een paar van haar posts geÃ«mbed en langs elkaar gezet. Daarna heb ik met gebruik van een caroussel effect gezorgd dat je door de drie posts kan scrollen. Dit alles is gemaakt in Javascript. Je kan ook interacten met een post als deze vooraan staat van de caroussel. Zo kan je een video afspelen of naar de post gaan.

Music

Op de muziek pagina heb ik Ruby Mus haar meest recente muziek gezet. We waren van plan om met pijltjes door haar muziek te kunnen scrollen, maar helaas heeft ze pas 1 nummer uitgebracht. Door op de play knop te drukken begint een preview van het liedje te spelen en begint onder in de container een equalizer animatie. Verder beweegt het bolletje op de tijdlijn ook en word de tijd aan beide kanten geupdate. In deze pagina zit redelijk veel Javascript verwerkt.

Wat heb ik geleerd

Ik heb veel ervaring voor vooral Javascript opgedaan, maar daarnaast ook meer over HTML en CSS geleerd. Ook heb ik voor het eerst gewerkt met een simpele javascript library.

Wat kon beter

Ik had me meer kunnen focussen op het responsive maken van de website. Zo is de website niet erg aansprekelijk op kleinere beeldschermen.
