Slideshow

In dit project heb ik met gebruik van HTML, CSS en Javascript een slideshow gemaakt. Hierin heb ik vier foto's laten afwisselen. In dit project het ik met redelijk weinig ervaring veel geleerd.

Klik hier om het project te zien Github

Development

HTML

Het HTML deel in dit project is kort en simpel. Ik heb een container gemaakt voor de foto's, waarin ik vier foto's in gezet heb. Verder heb ik mijn stylesheet en javascript file gelinkt.

CSS

In CSS heb ik de stijl toegevoegd. Ik heb de foto's position: absolute gegeven, zodat deze over elkaar heen staan. Verder heb ik de container net zo groot gemaakt als het scherm, en heb ik de afbeeldingen in het midden gezet. Onderaan heb ik classes toegevoegd voor de foto's. Dezen worden aangeroepen vanuit Javascript. Hier zit een transition van 2s op. Zo heb ik de foto's in elkaar over laten lopen.

JS

In de Javascript code heb ik gebruik gemaakt van vier loop functies, een per foto in de slideshow. In het begin roep ik een keer de eerste functie aan. In de functies geef ik de foto een class. Deze class zorgt ervoor dat de foto zichtbaar word. Ik verander de opacity via CSS, zodat ik makkelijk een transition kan toevoegen. Nadat de class word gelinkt word er 2000ms gewacht. Daarna word de class weer van de foto afgehaald. Ondertussen word de tweede foto zichtbaar enzovoort.

Wat heb ik geleerd

Ik heb meer geleerd over Javascript. Zo heb ik gebruik gemaakt van classlist.toggle, en heb ik gebruik gemaakt van een sleep function die ik nog veel ga gebruiken in andere projecten.

Wat kon beter

Ik had me beter in de code kunnen verdiepen om slimmere oplossingen te vinden. Zo kon de code geminimaliseerd worden.
