Rock-Paper-Scissors

Een van mijn eerste projecten binnen development was het maken van een rock - paper - scissors game. Met weinig ervaring heb ik deze game werkend gemaakt met javascript, en de stijl geprogrammeerd in HTML/CSS. Met dit project heb ik de basics van javascript geleerd.

Klik hier om de game te spelen Github

Development

HTML

Om de layout voor de game te maken heb ik eerst 3 buttons en wat tekst voor het resultaat gemaakt in HTML. Daarna heb ik images toegevoegd die zichtbaar worden als de bijbehorende knop word ingedrukt. Om dit te doen heb ik ze eerst allemaal onzichtbaar gemaakt en position: absolute gegeven, zodat ze eigenlijk over elkaar staan. De images staan aan zowel de player als enemy kant, alleen worden de images aan de enemy kant niet zichtbaar door buttons, maar door een script die de keuze van de enemy random bepaald.

CSS

De stijl van de game heb ik met CSS toegevoegd. Omdat ik toen ik dit project maakte niet veel regels code voor de stijl had verwacht, heb ik het in de HTML file gezet. Met de kennis die ik nu heb weet ik dat dit onoverzichtelijk kan worden met grotere projecten. De stijl voor de game is redelijk simpel, omdat ik de focus op het werkend maken van de game had gelegd. Dus met wat simpele lines CSS heb ik de game wat duidelijker gemaakt.

JS

Om de game werkend te maken heb ik javascript gebruikt. De eerste stap was kijken welke knop er word ingedrukt. Op basis daarvan word er een van de drie icoontjes zichtbaar. Daarna word de uitkomst van de tegenstander gegenereerd. dit is een nummer tussen 0 en 2. Aan de knoppen zitten ook waardes tussen 0 en 2. Zo kan ik de waardes van de knoppen en de enemy vergelijken en uitslagen van de game trekken. Ook zit er een reset functie in, om ervoor te zorgen dat de game niet raar gaat doen als het vaker achter elkaar word gespeeld. Uiteindelijk werkt het zoals ik wilde.

Wat heb ik geleerd

Ik heb geleerd om gebruik te maken van simpele Javascript in dit project.

Wat kon beter

Ik had meer focus op de code kunnen leggen om hiervoor slimmere oplossingen te vinden. Verder had ik de stijl in een apart document kunnen zetten.
