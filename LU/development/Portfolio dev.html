Portfolio Dev.

Door het hele semester heen heb ik veel aan mijn portfolio gewerkt. Een groot deel hiervan was het design omzetten in code. Zo heb ik met gebruik van HTML, CSS en javascript mijn design omgezet in een werkende website. In dit project laat ik mijn proces hiervan zien, en focus ik op de interresante development delen uit mijn portfolio.

Github

Development

HTML

Mijn portfolio is onderverdeeld in drie pagina's: Homepagina, leeruitkomsten pagina en projecten pagina. Voor de leeruitkomten en projecten pagina's heb ik eerst het design uitgewerkt in een HTML template. Deze kon ik makkelijk dupliceren en vullen met andere content. Zo heb ik uiteindelijk 1 homepagina, 6 leeruitkomsten pagina's en ongeveer 25 project pagina's.

CSS

Om stijl toe te voegen aan de pagina's heb ik een groot style.css bestand aangemaakt. Hierin staat de stijl van alle pagina's. Door het gebruik van duidelijke ID's en classes in HTML is het bestand, ondanks de 800 regels, redelijk overzichtelijk. Daarnaast is er nog een apart CSS bestand voor de intro van de homepagina. In beide bestanden heb ik gebruik gemaakt van verschillende keyframes, om animaties toe te voegen als details. Denk aan de intro van de homepagina of de animatie wanneer er word gehovered over een project.

JS

Om bepaalde elementen werkend te maken heb ik ook javascript gebruikt. Zo heb ik een javascript bestand gemaakt om tussen de leeruitkomsten op de homepagina te scrollen met pijltjes. Ook word de progressie weergegeven bij de cirkels onder de leeruitkomsten. Daarnaast is er ook een javascript bestand voor de custom pointer. Hierin word geregeld dat het object de muis volgt, maar ook dat deze animeert als er geklikt word. Daarnaast word ook het hover-effect over de projecten werkend gemaakt.

Wat heb ik geleerd

Ik heb meer ervaring opgedaan met het toepassen van HTML, CSS en javascript.

Wat kon beter

Mijn bestanden zouden beter gestructureerd kunnen worden. Ook zou ik meer CSS bestanden toegevoegd kunnen hebben.
