Leeruitkomst: Development Je programmeert en integreert proofs of concepts op basis van gevalideerde eisen en gedocumenteerd in een versiebeheertool.

Portfolio

Door het hele semester heen heb ik veel aan mijn portfolio gewerkt. Een groot deel hiervan was het design omzetten in code. Zo heb ik met gebruik van HTML, CSS en javascript mijn design omgezet in een werkende website. In dit project laat ik mijn proces hiervan zien, en focus ik op de interresante development delen uit mijn portfolio.

Rock-Paper-Scissors

Een van mijn eerste projecten binnen development was het maken van een rock - paper - scissors game. Met weinig ervaring heb ik deze game werkend gemaakt met javascript, en de stijl geprogrammeerd in HTML/CSS. Met dit project heb ik de basics van javascript geleerd.

Slideshow

In dit project heb ik met gebruik van HTML, CSS en Javascript een slideshow gemaakt. Hierin heb ik vier foto's laten afwisselen. In dit project het ik met redelijk weinig ervaring veel geleerd.

Website Ruby Mus

In dit semester heb ik veel tijd besteed aan onze werkgever Ruby Mus. Een groot deel hiervan is de website die we voor haar gemaakt hebben. De website is gemaakt in HTML, CSS en javascript.

Advanced development

In dit project heb ik onderzoek gedaan naar verschillende javascript libraries. Hiervan heb ik drie uitgewerkt in een demo. Uiteindelijk heb ik een hiervan ook in mijn portfolio verwerkt. Dit project was een van mijn eerste ervaringen met javascript libraries.

Sprint X

In mijn sprint X project heb ik veel focus gelegt op development. Mijn project was het maken van een tower defense game. Hierbij heb ik gebruik gemaakt van pure HTML, CSS en javascript. Zelf had ik al ervaring met game development, maar nooit heb ik hiervoor javascript gebruikt.
